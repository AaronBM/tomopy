correct_fov¶

tomopy.algorithms.preprocess.correct_fov.correct_fov(data, num_overlap_pixels=None)[source]¶

Stich 180-360 degree projections on 0-180 degree projections. This is usually the case when field of view of the detector is smaller than the object to be imaged.

Parameters:

data : ndarray

Input data.

num_overlap_pixels : scalar, optional

The overlapping regions between 0-180 and 180-360 degree projections.
Returns:

data : ndarray

Output processed data.
